Strapi Docker App

This project runs Strapi CMS using Docker and Docker Compose.
It allows quick setup of a Strapi environment without installing Node.js locally.

Getting Started
1. Build the containers
docker-compose build

2. Start the application
docker-compose up -d

3. Stop the application
docker-compose down


After starting, open:

http://localhost:1337

Running Strapi Manually (Optional)

If running outside Docker:

develop
npm run develop

start
npm run start

build
npm run build
